Meting

metingJS 是为Aplayer 添加网易云、QQ音乐等支持的插件 它可以获取其中的歌曲等

在线引用

github [API](Meting-API (injahow.cn))

常用参数

参数名 详细参数 参数说明
server netease、tencent、kugou、xiami、baidu 网易云音乐、QQ音乐、酷狗音乐、虾米音乐，百度音乐
type song、playlist、album、search、artist 歌曲、歌单、专辑、搜索结果、艺术家
id id值 每个歌曲、歌单都有自己的ID常在链接中
fixed true、false 默认false 为真时会生成一个吸附在页面左下角的播放器
mini true、false 默认false 为真时会生成一个 小播放器
theme pink 主题颜色
loop all、one、none 自动播放、单曲循环、无
order lists、random 顺序播放，随机播放
list-folded true、false 列表是否折叠 是、否
list-max-height 340px 列表最大高度
volume 0 - 1 音量
